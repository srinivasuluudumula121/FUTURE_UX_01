# User Flow

## Primary User Journey

User visits website
→ Understand clinic offering
→ Explore treatments
→ Build trust through testimonials and credentials
→ View service details
→ Contact clinic or book appointment

## Homepage User Intent

Most users arrive to:

* Understand treatments
* Check clinic credibility
* Compare services
* Contact clinic quickly
* Book consultation

## Appointment Flow

Homepage CTA
→ Contact / Appointment Page
→ Submit inquiry
→ Clinic contact follow-up

## Mobile Experience

The website was designed responsively to ensure:

* Easy navigation
* Fast scanning
* Touch-friendly CTA buttons
* Better appointment accessibility

# Design System

A modern healthcare-focused design system was created to maintain visual consistency across the redesigned Partha Dental Clinic website.

The visual language was inspired by premium modern healthcare websites while preserving Partha Dental Clinic’s original brand identity and colors.

The system was designed to support both desktop and mobile responsiveness.

## Design Direction

The interface focuses on:

* Professional healthcare feel
* Trust and reassurance
* Clean visual hierarchy
* High readability
* Conversion-focused interactions
* Responsive usability

## Color System

The redesign uses Partha Dental Clinic’s existing brand colors to maintain familiarity and strengthen brand recognition.

### Primary Brand Colors

Used for:

* Call-to-action buttons
* Appointment booking sections
* Interactive states
* Navigation highlights

### Neutral Colors

Used for:

* Backgrounds
* Cards
* Supporting sections
* Readability balance

### Accent Colors

Used for:

* Highlights
* Informational emphasis
* Supporting UI elements

## Typography

Typography was structured to improve readability and scanning.

The system prioritizes:

* Strong heading hierarchy
* Clear service readability
* Better section separation
* Mobile readability

Typography levels include:

* Hero headings
* Section titles
* Card headings
* Body content
* Supporting labels

## Components

Reusable UI components were created for consistency.

Components include:

* Navigation bar
* CTA buttons
* Service cards
* Trust cards
* Testimonial cards
* Contact forms
* Appointment sections
* Footer components

## Spacing & Layout

A consistent spacing system was used to improve readability and maintain visual rhythm across both desktop and mobile layouts.

## Responsive Design

The design system supports:

### Desktop Experience

Focused on:

* Spacious layouts
* Clear hierarchy
* Rich content presentation

### Mobile Experience

Focused on:

* Touch-friendly interaction
* Faster scanning
* Sticky CTA visibility
* Reduced friction

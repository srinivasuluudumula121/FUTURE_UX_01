# Partha Dental Clinic – Website Redesign

UI/UX Design Project for Future Interns – Task 01

---

## Project Overview

This project is a conversion-focused website redesign for **Partha Dental Clinic**, created as part of Future Interns UI/UX Design Task 01.

The goal of this redesign was to improve website clarity, build trust with first-time visitors, and increase appointment conversion through better UX, stronger visual hierarchy, and modern responsive design.

The redesign maintains the clinic’s original brand identity while improving usability and lead generation.

---

## Problem Statement

Many local service business websites struggle with:

- unclear messaging
- poor visual hierarchy
- weak calls-to-action
- poor mobile responsiveness
- information overload

As a result, visitors often leave without contacting the business or booking appointments.

This redesign solves those challenges by creating a cleaner, conversion-focused website experience.
